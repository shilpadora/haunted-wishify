# Product Overview

## Haunted Wishify Designer & Spooky Web Builder

This is a dual-purpose Halloween-themed creative platform consisting of:

### 1. Haunted Wishify Designer (Primary App)
A spooky greeting card creator with multi-page book functionality:
- Interactive template-based card design system
- Real-time visual editor with drag-and-drop components
- Multi-page book creation with PDF export
- Halloween, Gothic, Birthday, Valentine, Christmas, and Anniversary themes
- Animated effects (floating ghosts, confetti, hearts, snow, mystical elements)
- Professional export options (PNG, PDF, print-ready formats)

### 2. Spooky Web Builder (Secondary App)
A Halloween-themed drag-and-drop website builder:
- 1995 retro aesthetic with glitch effects and haunted animations
- Specialized landing page builder with 30-second opening sequences
- Retro cassette music player with ghost sounds
- Component library with ghostly headers, haunted buttons, spectral forms
- Immersive audio integration with supernatural sound effects

## Target Experience
Both applications emphasize atmospheric, immersive user experiences with:
- Consistent spooky theming (blacks, oranges, purples, greens with glowing effects)
- Supernatural animations and particle effects
- Audio integration for enhanced atmosphere
- Professional export capabilities
- No-code/low-code approach for non-technical users

## Key Value Propositions
- Create professional-quality spooky content without design skills
- Immersive Halloween-themed creative tools
- Multi-format export capabilities
- Browser-based, no installation required
- Atmospheric audio and visual effects
